## The Core Risk-Adjusted Return Metrics

Sharpe Ratio = (Portfolio return - Risk-free rate) / Portfolio standard deviation of returns.

Using a risk-free rate of approximately 5% (US T-bill yield at current rates) and calculating weekly return standard deviation from the position log (Article 279, Tab 8), the Sharpe ratio measures how much excess return is generated per unit of volatility.

Target: Sharpe ratio above 1.2 consistently. Above 1.5 is excellent. Below 0.8 suggests risk is being taken without sufficient return compensation.

Sortino Ratio = (Portfolio return - Risk-free rate) / Downside deviation (only negative return weeks in the denominator).

The Sortino ratio is more appropriate for commodity portfolios where upside volatility (large winning weeks) should not be penalized alongside downside volatility. Target: Sortino above 1.8.

Maximum Drawdown = Peak-to-trough account value decline during the measurement period.

The maximum drawdown should be tracked from the rolling all-time equity peak. When maximum drawdown exceeds 15%, the two-speed drawdown protocol activates (Article 160). Annual maximum drawdown target: below 15%.
