## The Sharpe Ratio: Return Per Unit of Risk

The Sharpe ratio measures excess return (returns above the risk-free rate) divided by return volatility. A Sharpe ratio above 1.0 indicates that each unit of risk taken generated more than one unit of return above the risk-free rate. A ratio below 0.5 indicates poor risk-adjusted performance regardless of absolute return level.
