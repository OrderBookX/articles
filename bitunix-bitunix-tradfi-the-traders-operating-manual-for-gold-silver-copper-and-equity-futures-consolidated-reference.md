## Pre-Trade Checklist (Complete Before Every Entry)

1. Current macro regime identified: boom / goldilocks / stagflation / bust
2. Regime alignment confirmed: this instrument is favored by current regime
3. Primary driver signal checked: TIPS yield for XAUUSDT, PMI for COPPERUSDT, delivery data for TSLAUSDT
4. Technical confirmation present: breakout, bounce from support, or tested breakout
5. Risk-reward ratio calculated: entry price, stop-loss price, target price, ratio minimum met
6. Position size calculated: using two-step method (maximum loss / stop distance = position size)
7. Funding cost estimated: rate x expected holding period x leverage = total funding cost
8. Stop-loss order ready: to be placed simultaneously with entry, never after
9. Thesis invalidation condition defined: one sentence, specific, measurable
10. Journal entry written: instrument, direction, macro context, fundamental driver, technical setup, risk parameters, invalidation condition
